# Glossary

| Term | Meaning |
|---|---|
| **ADS-B** | Automatic Dependent Surveillance-Broadcast. The signals airplanes transmit with their position. Picked up by ground receivers and aggregated by airplanes.live / ADSB.lol. |
| **APOD** | Astronomy Picture of the Day — NASA's daily astronomy photo + explanation. |
| **CO-OPS** | NOAA Center for Operational Oceanographic Products and Services. Provides US tide-station data. |
| **EMSC** | European-Mediterranean Seismological Centre. Seismic event aggregator. |
| **EONET** | NASA's Earth Observatory Natural Event Tracker. Categorizes natural events (fires, volcanoes, storms, ice). |
| **GDACS** | Global Disaster Alert and Coordination System. Provides alert levels (Green/Orange/Red) for active disasters. |
| **GeoNet** | New Zealand's geological hazards monitoring network. Provides high-resolution NZ-area seismic data. |
| **Hex** (in aircraft) | The ICAO24 transponder address — uniquely identifies an airframe (6 hex chars). |
| **ICAO** | International Civil Aviation Organization. Assigns the unique 24-bit codes carried in ADS-B. |
| **MLLW** | Mean Lower Low Water — NOAA's reference datum for tide levels in the US. |
| **MMI** | Modified Mercalli Intensity — perceived shaking intensity scale for earthquakes. |
| **MOC** | Map of Content — an Obsidian convention for index notes that link to related notes. See [[00 Master Index]]. |
| **NHC** | National Hurricane Center. NOAA. Active tropical cyclones. |
| **Nixpacks** | Railway's auto-detected build system. Reads `nixpacks.toml` to configure language + dependencies. |
| **NWS** | National Weather Service. NOAA. Active weather alerts via `api.weather.gov`. |
| **OpenSky** | An open-source ADS-B aggregator. Was our aircraft source, replaced by airplanes.live due to rate limits. |
| **Palantir-style** | The visual aesthetic — tight grids, corner ticks, monospace numerics, dark with cyan/magenta accents, intel-grade density. |
| **POE2** | Path of Exile 2 — the ARPG from Grinding Gear Games. The game the gaming-videos rail is filtered to. |
| **PWA** | Progressive Web App. We don't ship a manifest, but the dashboard would PWA-install fine. |
| **RSS / Atom** | XML feed formats. We parse both. YouTube channel feeds are Atom. |
| **SWPC** | NOAA Space Weather Prediction Center. |
| **TLE** | Two-Line Element set — the standard 69-character × 2 format for satellite orbital elements. Propagated to current sub-points via `satellite.js`. |
| **TTL** | Time-to-live. Cache expiration time in seconds. |
| **TTS** | Text-to-Speech. The AI Voice agent. See [[Features/AI Voice Agent]]. |
| **USGS** | US Geological Survey. Primary US earthquake source. |
| **Webview** | Our slide-out iframe for opening source pages without leaving the dashboard. See [[Features/Webview Slide-out]]. |
